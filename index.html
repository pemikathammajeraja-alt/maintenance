Slack Webhook — แจ้งเตือนอัตโนมัติ

1
เปิดเว็บ Slack API
ไปที่ api.slack.com/apps แล้ว Login ด้วยบัญชี Slack ของบริษัท
2
สร้าง App ใหม่
กด Create New App → เลือก From scratch → ตั้งชื่อ เช่น "แจ้งซ่อม" → เลือก Workspace บริษัท → กด Create App
3
เปิด Incoming Webhooks
ในเมนูซ้าย → Incoming Webhooks → สลับสวิตช์เป็น On → กด Add New Webhook to Workspace → เลือก channel → Allow
⚠️ ถ้ายังไม่มี channel ให้เปิด Slack สร้าง channel ชื่อ #maintenance ก่อน แล้วค่อยกลับมา
4
Copy URL มาใส่ด้านล่าง
URL จะขึ้นต้นด้วย https://hooks.slack.com/services/...
✅ บันทึก Slack URL แล้ว
💡 กด "ทดสอบ" หลังใส่ URL — ถ้า Slack ได้รับข้อความ "ทดสอบระบบแจ้งซ่อม" แสดงว่าเชื่อมได้แล้ว
📊

Google Sheets — บันทึกข้อมูลถาวร

1
สร้าง Google Sheet ใหม่
ไปที่ sheets.google.com → กด + สร้างไฟล์ใหม่ → ตั้งชื่อ เช่น "บันทึกการซ่อม 2025"
2
เปิด Apps Script
คลิกเมนู Extensions (ส่วนขยาย) → Apps Script → จะเปิดหน้าใหม่
3
วาง Code นี้ แล้ว Save
ลบ code เดิมออกทั้งหมด → วาง code ด้านล่าง → กด 💾 Save
function doPost(e) { var sheet = SpreadsheetApp.getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.id, data.datetime, data.reporter, data.dept, data.machine, data.type, data.priority, data.desc, data.status, data.images || '' ]); return ContentService.createTextOutput('OK'); }
4
Deploy → Web App
กด Deploy (มุมขวาบน) → New deployment → Type เลือก Web app → Who has access เลือก Anyone → Deploy → กด Allow
⚠️ ถ้าระบบขอ "Review permissions" ให้กด Allow ได้เลย เพราะเป็น script ของเราเอง
5
Copy Web App URL มาใส่ด้านล่าง
✅ บันทึก Sheets URL แล้ว
